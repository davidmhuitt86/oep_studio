# SDD-011

# Property Inspector

Version: 1.0

Status: Draft

---

# Purpose

The Property Inspector displays editable information for the currently selected engineering item.

It provides a consistent editing experience across all Studios.

---

# Display

The Property Inspector updates automatically when selection changes.

Only one object is displayed at a time.

---

# Sections

Version 1 supports:

* General
* Metadata
* Tags

Future Studios may extend with discipline-specific sections.

---

# Behavior

Changes shall not be written immediately.

Users shall explicitly Save or Cancel modifications.

---

# Validation

Foundation performs all validation.

Studio displays validation messages.

---

# Engineering Principle

Editing shall remain predictable and reversible.
